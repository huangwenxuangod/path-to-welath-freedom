技术栈：TypeScript + TSX + React + Next.js (最新版 App Router, SSR) + Hero UI + Tailwind CSS (默认配置)

包管理器：Bun
代码格式化：Biome.js
部署平台：Vercel

状态管理：Zustand (全局状态) + TanStack Query (数据获取与缓存)
表单处理：React Hook Form + Zod

后端服务：
- Supabase (数据库)
- Clerk (Auth 认证)
- Resend (邮件营销管理)
- Stripe (海外支付：一次性、月费、年费订阅)
- UploadThing (文件上传)
- RevenueCat (收益分析)
- PostHog (数据分析)

API 路由：Next.js Route Handlers (与 Supabase Edge Functions 可共同使用)

测试：Vitest

动画：Framer Motion
图标：Lucide React

移动端：React Native + Expo Go + NativeWind + Lucide React
浏览器插件：WXT + React + TypeScript + Hero UI + Tailwind CSS + Lucide React

项目结构：全部按类型组织 (components、hooks、lib、types 等)，不按功能模块组织。
