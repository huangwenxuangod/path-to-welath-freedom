# 保姆级教学-Openclaw接入Discord&飞书

上一篇文章教大家如何用 Telegram 打造龙虾军团，这一期来分享一下 **Discord** 和 **飞书** 的部署方式，手把手带大家把这两个平台也接入 Openclaw。

如果有不懂的，可以看一下上篇文章

> 3月5日

本文分为两个部分：

- **第一部分：Discord 接入**
- **第二部分：飞书接入**

如果只想看飞书的朋友可以直接跳转到第二部分。

## 第一部分：Discord 接入

相比 Telegram，Discord 有三个明显优势：

1. **UI 和交互界面更好看**
2. **Bot 之间可以互相通信**——在群里可以互相 @
3. **更强的会话管理能力**——非常适合多个 Bot 协同处理

第一步：创建 Discord 服务器

打开 Discord，新建一个服务器：

![图像](https://pbs.twimg.com/media/HC2zS-FbwAAGgC6?format=jpg&name=large)

然后依次点击：**亲自创建** → **仅供我和我的朋友使用** → 输入名字

![图像](https://pbs.twimg.com/media/HC2zZjVbMAAegre?format=jpg&name=large)

第二步：创建机器人并获取 Token

打开 Discord 开发者后台：[https://discord.com/developers/applications](https://discord.com/developers/applications)

登录后点击 **创建新的 APP**，输入名字：

点击左侧的 **Bot**，点击 **重置令牌**，把这个 Token 复制下来备用：

![图像](https://pbs.twimg.com/media/HC2zkJEbYAAQJ3r?format=jpg&name=large)

把下面三个权限打开，然后点击 **保存更改**：

![图像](https://pbs.twimg.com/media/HC2zvoRbcAANx1R?format=jpg&name=large)

第三步：配置 OAuth2 权限

![图像](https://pbs.twimg.com/media/HC2z0BWakAAt3lW?format=jpg&name=large)

选择左侧的 **OAuth2**，勾选右侧的 **bot**，然后往下滑动：

勾选机器人权限中的 **发送消息** 和 **阅读历史消息**：

![图像](https://pbs.twimg.com/media/HC2z4I6bIAAsTAt?format=jpg&name=large)

第四步：将机器人添加到服务器

滑到最底下会看到一个 URL 链接，复制出来在浏览器中打开。如果你有客户端会跳转客户端，没有则在网页里选择服务器：

![图像](https://pbs.twimg.com/media/HC20BbbaQAAh1f7?format=jpg&name=large)

选择服务器，点击 **授权**，就能看到服务器里多了一个机器人：

![图像](https://pbs.twimg.com/media/HC20LFsbYAAS4Bh?format=jpg&name=large)

第五步：终端配置 Openclaw

回到终端，输入命令：

openclaw onboard

进入初始化页面后，按以下步骤操作：

1. **选择 yes**（用方向键向左，默认是 no）
2. **选择 QuickStart**如果是第二次配置，选择 Use existing values
3. **选择模型**如果已选好模型，按方向键上选择 Skip for now，然后选 All providers → Keep current 推荐使用：Gemini、ChatGPT 或 Claude 这里选择 OpenAI，方向键选择后按回车确认 选择 Codex 进行认证 **注意**：需要将 VPN 开启 TUN 模式
4. **选择通讯频道**选择 **Discord**（第三个选项），按回车 粘贴刚才保存的 Bot Token 选择 yes 选择 Open
5. **选择 Skills**：选择 no，后面手动安装
6. **填写 API Key**：全部选择 no
7. **Enable hooks?**：选择 Skip for now，按空格再按回车如果已经安装过，这一步会让你重启，重启即可
8. **选择 UI 页面**：选择 Open the Web UI

第六步：配对验证

回到 Discord 发一句消息，你会看到一条类似这样的命令：

openclaw pairing approve discord YWVxxxx

在终端执行这条命令即可完成配对：

![图像](https://pbs.twimg.com/media/HC20SAPaAAAUko9?format=jpg&name=large)

多 Agent 配置

如果你需要多个 Agent，只需多注册几个 Bot Token，重复上面的操作。然后告诉 AI：我需要多配置几个 Agent，每个 Agent 负责什么，Token 分别是什么。

具体提示词可以参考上一篇文章，让 AI 改成 Discord 版本即可。

## 第二部分：飞书接入

之所以接入飞书，是因为这个软件对国人最友好。但配置相对麻烦一些，大家需要仔细操作。

第一步：创建飞书应用

打开 [飞书开放平台](https://open.feishu.cn/app)，没有飞书的需要先注册，然后创建 **企业自建应用**：

![图像](https://pbs.twimg.com/media/HC20Vvua8AAsl-r?format=jpg&name=large)

第二步：获取凭证

点击左侧的 **凭证和基础信息**，复制右边的 **App ID** 和 **App Secret**：

![图像](https://pbs.twimg.com/media/HC20Y3AawAAVv5X?format=jpg&name=large)

第三步：添加机器人能力

点击 **添加应用能力**，然后点击 **机器人** ：

![图像](https://pbs.twimg.com/media/HC20epmaYAEQkTc?format=jpg&name=large)

第四步：配置权限

点击左侧的 **权限管理**，按需添加以下权限：

- 事件订阅
- 通讯录
- 消息与群组
- 云文档
- 多维表格

> 省事的话可以直接点左上角的 **全选** **注意一定要先添加机器人能力再进行配置权限不然是没有接受消息和发送消息的权限的**。

![图像](https://pbs.twimg.com/media/HC20kZRboAAaecu?format=jpg&name=large)

第五步：发布应用

![图像](https://pbs.twimg.com/media/HC20zBIbsAAlT_m?format=jpg&name=large)

选择 **版本管理与发布**，填个版本号（比如 1.0.0），保存发布：

第六步：终端配置 Openclaw

回到终端执行：

openclaw onboard

> 小技巧：输入 openclaw o 然后按 Tab 键可以自动补全，不用记住完整命令。

按以下步骤操作：

1. **选择 yes**（用方向键向左，默认是 no）
2. **选择 QuickStart**如果是第二次配置，选择 Use existing values
3. **选择模型**如果已选好模型，按方向键上选择 Skip for now，然后选 All providers → Keep current 推荐使用：Gemini、ChatGPT 或 Claude 这里选择 OpenAI，方向键选择后按回车确认 选择 Codex 进行认证 **注意**：需要将 VPN 开启 TUN 模式
4. **选择通讯频道**选择 **Feishu/Lark**（第九个选项），按回车 选择 Download from npm 安装插件 输入你的 **App Secret** 输入你的 **App ID** 选择 **WebSocket** 选择国内飞书还是海外版飞书 选择 Open
5. **选择 Skills**：选择 no，后面手动安装
6. **填写 API Key**：全部选择 no
7. **Enable hooks?**：选择 Skip for now，按空格再按回车如果已经安装过，这一步会让你重启，重启即可
8. **选择 UI 页面**：选择 Open the Web UI

第七步：配置事件订阅

回到飞书开放平台，点击左侧的 **事件与回调** → **事件配置**，然后点击订阅方式，选择 **长连接** 接收事件，保存：

![图像](https://pbs.twimg.com/media/HC207siboAA2izF?format=jpg&name=large)

点击 **添加事件**，选择 **消息与群组**，按需勾选事件：

![图像](https://pbs.twimg.com/media/HC21JqSbEAAfmwV?format=jpg&name=large)

和之前一样创建版本并发布。

第八步：打开应用并验证

![图像](https://pbs.twimg.com/media/HC21QqRbQAEdqKB?format=jpg&name=large)

回到飞书，点击打开应用：

随便发送一条消息，会提示你执行一个放行命令，在终端执行即可：

![图像](https://pbs.twimg.com/media/HC21ZdRasAAu0GT?format=jpg&name=large)

到此，Discord 和飞书两个平台都接入完毕了。下课，开始愉快地养虾吧！
