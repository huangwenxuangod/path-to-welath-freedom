# 我花49块买了李一舟的课，差点又掏了1980，最后是Claude救了我

我险些花1980买了李一舟的课：一次关于真诚、流量与自我认知的思考

两天前的早上，我无聊刷直播时，偶然刷到了李一舟的直播课。初听5分钟，感觉讲得很实在。他讲的话题正是我感兴趣的：**一人公司、超级个体、做IP，以及如何利用AI提效、教小白学习智能体等等。**

恰好我最近和另一个社区要推出智能体课程，我想学习他是如何用通俗的语言讲智能体的。

于是我买了他的三天课程，49块钱。

购买后，助教立即加我微信，每天晚上预告直播链接。这个链接进入的是他们公司自己的小程序直播间——一个私域空间。

前两天我一直没空，也没上课。今天是第三天，因为课程没有回放，错过就错过了。想着已经花了钱，一节课不听也可惜，于是硬着头皮去看了。

**没想到，讲的干货还是挺多的。**

两个小时的时间，我边听边记，收获颇大。而且，我差点就下单购买他1980元的年费课程。

声明：我对他的年费课程本身不做评价，因为我没看过。我只看了三天课程中的其中一天。但就这一天而言，以及就我险些下单而言，我觉得他的转化效果做得非常不错。

我一边听一边记的笔记（部分）

**一、 李一舟的“地下”生意经**

购买后，助教立马加微信，拉入私域。前两天我忙得没空看，直到第三天，想着钱都花了，硬着头皮点开了那个私域小程序的直播链接。

我发现了一个有趣的现象： 在公域（抖音/视频号）上，你很难再看到他卖 1980 元的高价课了。之前的风波让他学会了“低调”。

现在的玩法是：**公域只卖 49 或 99 的低价课引流，把人筛进私域小程序后，再进行高客单价的转化。** 在这里，他可以卖 1980，甚至 19800，极大降低了在公域被骂、被封的风险。

这套“公域放饵，私域收网”的逻辑，被他玩得炉火纯青。

**二、 49块钱里的“真干货”**

抛开商业模式不谈，单就这第三天的课程内容，我有三个很深的感触。

**1\. 为什么周鸿祎要自己剪视频？--关于外包的边界**

李一舟提到一个观点：**“很多重要的事情，只能自己做。”** 他举例周鸿祎，身家如此之高，依然坚持在剪映上自己剪视频，哪怕剪得很粗糙。 为什么？是为了省钱吗？当然不是。 **是为了保持对流量的“体感”和“敏感度”。**

这点让我非常震撼。作为超级个体，我们总想着自动化、想外包。

但有些东西是不能外包的。 AI 可以帮我选题、润色、生成素材，但**“最终的决策”**和**“对内容的网感”**，必须掌握在自己手里。一旦脱离了一线实操，这种敏感度就会退化。

**2\. 成年人的核心课题：寻找“天命”**

每个人都有流量天赋，只是很多人没看见自己。

罗振宇在跨年演讲里说过：要找到一份只有你能做的工作。

李一舟也强调，做 IP 乃至过这一生，最重要的就是找到那个**“由你独一无二的经历锻造出的天赋点”**。

找不到这个点，再多的技巧也是徒劳；找到了，剩下的就是坚持。

**3\. 状态 > 形式**

做内容，真实的情感状态远比打光、布景重要。 如果你此刻有表达欲，不要管背景乱不乱，不要管发型好不好，立马拿起手机拍。

**形式是给外行看的，情绪才是给人心看的。**

**三、 险些被“割”，是 AI 救了我**

李一舟最厉害的天赋，确实是**“真诚的某种变体”**——他拥有一种极强的说服力。

听了两个小时，我边听边记，甚至一度产生了冲动，想下单他 1980 元的年费课程。 那种氛围的烘托，那种对痛点的精准打击，让我差点就点了支付。

但在最后关头，我做了一个动作。 我没有问朋友，也没有问老婆，而是打开了我的 **Claude Code 、我的数字分身kevin\_AI**。

我让它结合我当下的状态、正在做的事情、以及李一舟课程的大纲，问它：“我需要买这个课吗？”

AI 看完之后给我生成了一个报告，告诉我不用买。于是我最终顶住了李大师的真诚宣传，收手了。

**这也是这个时代最魔幻的地方：** 我花钱去听一个人教我用 AI，最后是 AI 告诉我，别花这笔冤枉钱。
