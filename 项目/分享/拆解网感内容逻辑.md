# 二、开篇能力训练法（最值钱的能力）

你说的完全正确：

**爆款 = 开篇定生死。**

## 你的训练方法（我帮你结构化）

1. **看到吸引你的开头，立刻抄下来**
2. **分析三件事**
    
    - 用了什么**情绪**？
    - 用了什么**结构**？
    - 用了什么**关键词**？
    
3. **仿写**：换主题，套它的句式
4. 做多了 → **形成肌肉记忆**

## 开篇万能结构（你以后套就行）

1. **反常识**
    
    大多数人以为… 其实全错了。
2. **痛点扎心**
    
    你是不是也…
3. **结果前置**
    
    我靠这一招，…
4. **金句砸脸**
    
    我告诉你一个真相：…
5. **身份共鸣**
    
    所有想… 的人，一定要听。
