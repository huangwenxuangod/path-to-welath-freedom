# GitHub Paste Plugin 使用指南

> 让你的 VS Code 变成最强的 Markdown 写作神器。

## 🚀 为什么你需要它？
写博客/笔记最烦的是什么？**图床**。
你需要截图 -> 保存 -> 打开图床网站 -> 上传 -> 复制链接 -> 粘贴到 Markdown。
**现在，只需要一步：Ctrl + V**。

## ✨ 核心功能
*   **全自动上传**：支持图片、视频 (.mp4, .mov)、音频 (.mp3)、文档 (.pdf, .zip)。
*   **CDN 加速**：自动生成 `jsDelivr` 链接，全球访问速度飞快。
*   **智能识别**：自动判断剪贴板内容，是文本就粘贴文本，是文件就上传。
*   **零侵入**：专为 `.md` 文件设计，不影响写代码。

## 🛠️ 快速配置 (3步走)

### 第一步：准备 GitHub Token
1.  登录 GitHub，访问 [Settings > Developer settings > Personal access tokens](https://github.com/settings/tokens)。
2.  点击 **Generate new token (classic)**。
3.  勾选 **`repo`** 权限（这一步很关键，否则无法上传）。
4.  复制生成的 Token（切记！关掉页面就看不到了）。

### 第二步：安装与配置
1.  在 VS Code 插件市场搜索 `GitHub Paste Plugin` 并安装。
2.  打开 VS Code 设置 (`Ctrl + ,`)。
3.  搜索 `githubPastePlugin`。
4.  填入你的配置：
    *   **Token**: 刚才复制的那串字符。
    *   **Repo**: 你用来存图的仓库名（格式：`你的用户名/仓库名`，例如 `huangwenxuangod/assets`）。

### 第三步：开始爽
打开任意 `.md` 文件，截图，按下 `Ctrl + V`。
你会看到右下角提示“正在上传...”，几秒后，一张 CDN 加速的图片链接就出现在你的文档里了。

## ❓ 常见问题
*   **Q: 上传失败怎么办？**
    *   A: 检查 Token 是否过期，检查 Repo 名字是否写对（不要带 `https://github.com/`）。
*   **Q: 图片加载慢？**
    *   A: 插件默认使用 jsDelivr CDN，国内访问通常很快。如果慢，检查你的网络。
