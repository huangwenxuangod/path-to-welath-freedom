# GitHub Paste Plugin (GitHub 图片粘贴插件)

> **定位**：极致的 Markdown 写作体验优化工具。
> **核心卖点**：一键粘贴，自动上传 GitHub，秒变 jsDelivr CDN 链接。
> **适用场景**：技术博客写作、文档编写、笔记记录。

## 📂 文件夹结构
*   `教程/`: 存放插件的使用教程、配置指南。
*   `开发/`: 存放开发日志、版本迭代记录。
*   `素材/`: 存放演示 GIF、截图。
