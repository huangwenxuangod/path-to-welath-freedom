飞书最近出了面向开发者的cli，简单的来说，以后能够通过编程工具如claude code或者codex直接操作飞书去进行大部分的操作了，接下来我具体来介绍一下为什么要用这个cli，以及怎么去用。

首先为什么要用这个cli呢？之前爆火的openclaw由于权限安全问题大部分会被部署到云端，而claude code可以在本地跑，因此在同步本地知识库和飞书知识库的时候就会变得更加方便。

同时openclaw的过程不可见，但是编程工具的思考过程，命令输出都是实打实的，对于开发者来说会更加的友好。

下面我就来说一下怎么用，首先在终端命令复制粘贴这两行命令，直接运行。
