
最近claude code出了非常非常多的新功能，大有取代openclaw的势头，但是因为众所周知的原因，使用claude code实际上是会出现很多问题的，今天我就来讲讲我遇到过的坑。

先介绍一下claude code，是由Anthropic公司推出的AI编程助手，目前可以说是编程领域的老大了。但是要真正用上它可以说要经历九九八十一难。

第一关，安装难，这个问题可以说是相对来说最简单的了，此处可以直接查看下图的命令进行安装。

如果是windows的用户还需要进行下载git for windows，这里我推荐使用windows原生的winget命令直接进行安装

![[IMG-20260325195215416.png]]

`winget install --id Git.Git -e --source winget`

安装好之后，在终端里输入 claude 这个命令，就能启动交互了，这是成功后的图示。
![[IMG-20260325195215446.png]]

第二关，登录难，我是怎么发现这个问题的呢？那是因为有一次我照常打开claude准备编程的时候，弹出了这个错误界面。
![[IMG-20260325195215460.png]]

最后发现是claude 把我原来的google账号直接给封禁掉了,不是哥们，这你也知道？

![[IMG-20260325195215485.png]]

申诉+找回试了好几次也没行，于是我的选择是重开一个google账号。

之所以不用其他的邮箱账号是因为google可以说是支持claude比较好的了，这个被封了是因为其他原因所导致的。

1. ip地址随意切换，导致可能被检测到了。
2. 注册时的验证手机号和ip地址不对应，也有可能是手机号被重复使用了。

因此这次注册我选择打开了google无痕界面，先注册一个新的google邮箱，ip稳定在美国，接着就是验证手机号了，我选择的是5sim的美国虚拟手机号。

![[IMG-20260325195215514.png]]

在左侧面板选择claude，选择美国，接着点击购买，会弹出这样的界面，选择对应的虚拟手机号复制过去claude 那里，接码后即可。

![[IMG-20260325195215540.png]]

跳过会员选项后，就能看到主页面，就是登录成功了。

![[IMG-20260325195215572.png]]

第三关，使用难，虽然登录进来了，但是想用对话需要进行付费充值，而难点在于能够用于支付的卡是非常非常少的，有一种方案是通过apple id进行代充值，据说很稳定。

但对于我这样的windows用户来说，就不甚友好了，因此在这个阶段，我的选择是国产api进行替代，理论上用哪一家模型厂商都没有区别。

![[IMG-20260325195215604.png]]

具体来说是在环境编辑变量中替换这三个变量为你使用的模型的值，分别是认证需要用的token，请求发送到的地址，和需要使用的模型名这三样。

到这里基本上就能开始使用起来claude 的全部功能，当然模型能力会有所下降，但作为一个常年在使用vscode，trae的我来说，也算是吃上细糠了。

不断地学习，进步，提高。
